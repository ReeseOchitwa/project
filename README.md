<h1>Tic Tac Toe</h1>

<h2>Description</h2>
This project is a Tic Tac Toe game between two human players. It takes place in the console. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Pycharm</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
